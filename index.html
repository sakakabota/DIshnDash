Welcome to Dish-n-Dash

The Ultimate Robotic Dish Clearer

About the Project

Dish-n-Dash is a cutting-edge robotics project designed to automate waste sorting in school cafeterias. Using advanced object detection and robotics systems, it reduces contamination, streamlines recycling processes, and promotes sustainability.

Project Goals

  • Automate waste sorting to reduce human error.
  • Minimize waste contamination and optimize recycling efficiency.
  • Encourage eco-friendly behaviors in school settings.

How It Works

Dish-n-Dash uses a robust pipeline:

  1. Object Detection: Identifies dishes and their contents using a DETR-based model.
  2. Classification: Sorts waste into compost, recycling, or dish trays.
  3. Robotics Handling: Uses inverse kinematics to precisely manipulate dishes.

Challenges and Solutions

Throughout development, we faced key challenges:

  • Correctly tilting bowls for waste disposal.
  • Handling classification errors for certain object orientations.
  • Preventing hardware collisions caused by short USB-C cables.

Our innovative solutions included refining pose estimation algorithms and optimizing hardware configurations.

Future Improvements

To enhance Dish-n-Dash, we plan to:

  • Integrate "Grasp Anything" models for handling diverse objects.
  • Expand the classification model for more waste categories.
  • Implement failure detection and recovery systems.
