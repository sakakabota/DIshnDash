Introduction

An overview of the Dish-n-Dash project and its goals.

Project Overview

Dish-n-Dash is a robotic system designed to automate waste sorting in school cafeterias. This project aims to reduce waste contamination, lower operational costs, and promote eco-friendly habits among students.

Why This Project?

In school cafeterias, waste sorting is often done improperly, leading to recycling contamination and inefficiency. Dish-n-Dash addresses this issue using robotics and object detection to ensure accurate sorting and optimized waste management.

Real-World Applications

The principles and technology behind Dish-n-Dash have broader applications in waste management for public spaces, offices, and other institutions. The system also provides a platform for educational outreach, demonstrating how technology can address environmental challenges.
