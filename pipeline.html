System Pipeline

The Dish-n-Dash system follows a well-defined pipeline to achieve efficient waste sorting:

  1. Run Image Classifier Client: The classifier uses a transformer-based DETR model to detect and identify objects.
  2. Pick Up Bowl: The robot arm uses inverse kinematics to reach and pick up the identified bowl.
  3. Classify Contents: Based on the classification, contents are sorted into compost, recycling, or dish bins.

ROS Data Streams

Our system utilizes the following ROS components:

  • Server: Runs the classification model.
  • Nodes: Handles pose transformations and inverse kinematics execution.
  • Topics: Publishes end effector positions and classifications.
