Design Overview

Our robotic waste sorting system integrates computer vision, motion planning, and precise control to automate cafeteria waste management.

Design Goals

  • Accurate object classification and sorting
  • Safe and reliable robotic manipulation
  • Efficient waste stream separation
  • Real-time operation capability

System Requirements

  • Object detection accuracy > 95%
  • Processing time < 2 seconds
  • Safe handling of various materials
  • Proper waste stream segregation

Technical Implementation

Vision System

Our system uses a DETR (Detection Transformer) model with ResNet-50 backbone for object detection and classification:

  • Real-time object detection and classification
  • Custom-trained model for cafeteria items
  • Integration with ROS for robot control

ROS Architecture

Data Flow Pipeline

  • Client Node: Activates server and initiates classification
  • Server Node: Runs object detection model
  • Pose Transformation Node: Converts camera coordinates to robot space
  • Control Node: Executes motion planning and gripper control

Hardware Components

Sawyer Robot

  • 7 Degrees of Freedom
  • Integrated force sensors
  • Custom gripper attachment

Vision System

  • Intel RealSense Camera
  • Depth sensing capability
  • Wide field of view

Software Architecture

Key Components

  • ROS Noetic framework for system integration

  • Custom motion planning algorithms
  • Real-time control system
