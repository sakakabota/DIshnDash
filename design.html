Design

An in-depth look at the design choices and criteria for Dish-n-Dash.

Design Criteria

The primary design goal was to create a system capable of accurately sorting waste items with minimal human intervention. The following criteria were considered:

  • High accuracy in classifying waste categories (e.g., compost, recycling).
  • Seamless integration with existing cafeteria setups.
  • Durability and robustness to handle real-world use cases.
  • Cost-effectiveness for deployment in schools.

Design Overview

Our system consists of two main components:

  • Object Detection and Classification: Using a transformer-based object detection model (DETR with ResNet-50 backbone), the system identifies items such as bowls, fruits, and utensils with high precision.
  • Robotic Motion Planning: A robotic arm with inverse kinematics capabilities executes smooth trajectories to pick and sort items.

Key Design Choices

We made several important design decisions to meet our criteria:

  • Using a depth-sensing camera to enable 3D localization of objects.
  • Employing robust materials for the robotic arm to handle varying weights of items.
  • Optimizing the software pipeline to reduce latency in object detection and motion execution.

Challenges and Trade-offs

While formulating the design, we faced some trade-offs:

  • Balancing cost and accuracy: Advanced components like a depth camera and transformer model increased costs.
  • Simplifying grasp strategies: Instead of picking individual items, the robot focuses on handling bowls to minimize errors.
