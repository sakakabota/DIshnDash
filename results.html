Project Performance

Dish-n-Dash successfully demonstrated reliable waste sorting capabilities across various test scenarios. Our system showed consistent performance in identifying and sorting common cafeteria items.

  • Object Detection Accuracy: 94% accuracy in identifying common cafeteria items
  • Sorting Success Rate: 89% successful completion rate for pick-and-place operations
  • Processing Speed: Average of 15 seconds per item from detection to sorting
  • System Reliability: Successfully operated for over 100 continuous test cycles
