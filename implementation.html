DETR Server Setup

  • Tested Objects: apple, orange, banana, plastic spoon, plastic fork

Client Query Process:

1. Camera captures input

2. Image sent to DETR server

3. Server returns bounding box and centroid

4. Results processed for robot control

System Pipeline

Process Flow

  1. Image Acquisition and Processing
    • Camera captures scene image
    • Image preprocessed for detection
    • DETR server processes image
  2. Object Detection and Classification
    • Identify bowl and contents
    • Calculate object centroids
    • Determine object type (food vs utensil)
  3. Robot Control
    • Transform coordinates to robot space
    • Plan and execute motion
    • Sort items into appropriate bins

Technical Challenges

  • Server communication and reliability
  • Coordinate transformation accuracy
  • Object detection in varying positions
  • Integration of vision and motion systems
