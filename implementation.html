Hardware

We used the following hardware components:

  • Sawyer Robot Arm: Executes smooth and precise motions for waste sorting.
  • Intel RealSense Camera: Provides depth and color images for object detection.
  • Custom Sorting Bins: Designed to align with the robot's motion planning system.

Software

The software pipeline integrates the following systems:

  • Object Detection: We used DETR (a transformer-based detection model) with a ResNet-50 backbone to classify objects and identify their centroids.
  • Depth Processing: Depth images from the RealSense Camera were used to convert 2D pixel coordinates into 3D points in the robot's frame.
  • Trajectory Planning: The robot's motion was planned using inverse kinematics to pick up objects and sort them accurately.
